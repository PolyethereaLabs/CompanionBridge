# CompanionBridge

Transform your AI conversation history into comprehensive companion identity profiles with smart session analysis and optimized migration tools. Upload your interaction data, review conversations with memory anchor previews, and receive both detailed identity files and condensed token-efficient versions—plus ready-to-use JSON configurations for seamless AI platform migration.

## Features

- **Smart Conversation Processing**: Upload conversation JSON files up to 80MB
- **Memory Anchor Detection**: Automatically identifies key personality references 
- **Interactive Selection**: Review conversations with sortable tables and modal previews
- **Dual Output Formats**: Full detailed profiles and condensed transcript-free versions
- **Optimized Migration**: Ready-to-use JSON configurations for AI platforms
- **Advanced NLP**: spaCy-powered topic extraction and session analysis

## Deployment

This application is optimized for Replit deployment with:
- Flask web framework with gunicorn server
- PostgreSQL database support
- Secure file handling for large conversation datasets
- Bootstrap dark theme UI
